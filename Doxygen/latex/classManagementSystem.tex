Executa o algoritmo de Dijkstra para encontrar o caminho mais curto entre dois vértices. Complexidade temporal\+: O(\+E $\ast$ log(\+V)). 

Esta função calcula o caminho mais curto de um vértice de origem para um vértice de destino num grafo. Utiliza o algoritmo de Dijkstra para determinar o caminho e retorna a distância total desse caminho. O próprio caminho é armazenado no vetor fornecido.
